# Pinball3D
Using the Bevy game engine and Rapier physics engine for a simple 3D pinball game.

## Running the game with Rust
Rust and Cargo is a prerequisite.
```Bash
git clone https://github.com/gunstein/Pinball3D.git
cargo run --release
```
